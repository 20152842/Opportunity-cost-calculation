#### 시나리오 1: 마트 vs 편의점
```markdown
**상황**: 퇴근 후 우유를 사려는 직장인 (시급 20,000원)
- **선택지 A**: 가까운 편의점 (5분, 3,500원)
- **선택지 B**: 먼 마트 (25분, 2,800원)

**기존 판단**: "700원 차이니까 마트 가야지"
**앱 사용 후**: 
- A 총비용: 3,500 + (20,000÷60 × 5) = 5,167원
- B 총비용: 2,800 + (20,000÷60 × 25) = 11,133원
- **결과**: A가 5,966원 유리 → 편의점 선택
```

## 2. 문제점 진단

### 🔴 치명적 문제 (구조적)

#### 1) 사용자 여정(User Journey)이 없음
**현재**: "상황 → 계산 → 결과"만 표시
**부족**: 사용자가 **어떻게 앱에 도달**하고, **어떻게 사용**하고, **이후 무엇을 하는지** 없음

#### 2) 페인 포인트가 피상적
**현재**: "700원 차이니까 마트 가야지" (단순 계산 착오)
**부족**: 
- 왜 이 결정이 중요한가?
- 어떤 감정적 갈등이 있는가?
- 잘못된 결정의 결과는?

#### 3) 맥락(Context)이 부족
**현재**: "퇴근 후 우유를 사려는 직장인"
**부족**:
- 왜 지금 우유가 필요한가?
- 시간 압박이 있는가?
- 다른 선택지는 없는가?

```markdown
### 시나리오 1: 퇴근 후 장보기 딜레마

#### 👤 페르소나
**이름**: 김민준 (32세, 직장인)
**상황**: IT 기업 개발자, 야근 후 퇴근 (오후 9시)
**시급**: 20,000원 (연봉 4,200만원 기준)

#### 😰 페인 포인트 (문제 상황)

**9:00 PM - 퇴근 직후**
- 냉장고에 우유가 떨어짐
- 내일 아침 커피에 필요
- 집에서 도보 5분 거리 편의점 vs 25분 거리 대형마트

**내적 갈등**:
> "편의점은 가깝지만 비싸... 700원 차이면 마트 가는 게 맞지 않나?
> 근데 피곤한데... 25분씩이나 걸어가야 하나?
> 아니야, 700원이면 점심 한 끼 값인데. 그냥 마트 가자."

**기존 의사결정 방식**:
- 가격만 비교: 2,800원(마트) < 3,500원(편의점)
- 결정: 마트 선택
- 심리: 불확실함, "이게 맞나?" 싶은 기분

#### 🔍 사용자 여정 (User Journey)

**Step 1: 문제 인식 (집 앞)**
- 상황: 편의점 vs 마트 앞에서 고민
- 행동: 스마트폰 꺼내기
- 생각: "내 시간 가치는 얼마나 되지?"

**Step 2: 앱 접속 (웹 브라우저)**
- 행동: "시간 기회비용 계산" 검색 → 앱 발견
- 첫인상: "오, 딱 내가 필요한 거네"
- 소요 시간: 10초

**Step 3: 정보 입력 (앱 사용)**
```
[시급 입력]
- "내 시급이 얼마더라?" 고민
- 연봉 4,200만원 ÷ 12개월 ÷ 160시간 ≈ 21,875원
- "대충 20,000원으로 하자" 입력
- 소요 시간: 30초

[선택지 입력]
- 선택지 A: 편의점 (5분 걸어서, 3,500원)
- 선택지 B: 마트 (25분 걸어서, 2,800원)
- 소요 시간: 20초
```

**Step 4: 결과 확인 (계산 완료)**
```
결과 화면:
- ✅ 선택지 A(편의점) 권장
- 총비용: A = 5,167원 vs B = 11,133원
- 차액: 5,966원 절약

상세 계산:
📊 선택지 A (편의점)
・직접 비용: 3,500원
・시간 비용: 20,000원/60분 × 5분 = 1,667원
・총 비용: 5,167원

📊 선택지 B (마트)
・직접 비용: 2,800원
・시간 비용: 20,000원/60분 × 25분 = 8,333원
・총 비용: 11,133원
```
- 반응: "아, 시간까지 고려하면 편의점이 6,000원이나 싸네!"
- 감정: 확신, 안도
- 소요 시간: 10초

**Step 5: 의사결정 (실행)**
- 행동: 편의점으로 이동
- 심리: "이제 확신을 갖고 결정할 수 있어"
- 결과: 5분 만에 우유 구매 → 9:15 PM 귀가

**Step 6: 사후 평가 (다음 날)**
- 생각: "어젯밤에 마트 갔으면 9:40 PM 귀가였겠네"
- 추가 가치: 25분 절약 → 씻고 잘 시간 확보
- 만족도: ★★★★★
- 재사용 의도: "다음에도 이 앱 써야지"

#### 💡 페인 포인트 해결

| 페인 포인트 | 앱 사용 전 | 앱 사용 후 |
|------------|-----------|-----------|
| **불확실성** | "이게 맞는 선택인가?" 고민 | "확실히 편의점이 유리하다" 확신 |
| **감정적 갈등** | "아까운 700원..." 죄책감 | "6,000원 절약했다" 만족 |
| **의사결정 피로** | 매번 고민하며 에너지 소모 | 1분 만에 명확한 답 |
| **시간 가치 인식** | "내 시간은 공짜" 착각 | "내 시간은 분당 333원" 인식 |

#### 📈 측정 가능한 결과

**정량적**:
- 의사결정 시간: 5분 → 1분 (80% 감소)
- 총비용 절약: 5,966원
- 시간 절약: 20분

**정성적**:
- 의사결정 확신도: 40% → 95%
- 결정 후 후회: 있음 → 없음
- 재사용 의도: 90%
```

```markdown
### 시나리오 3: 대학생의 아르바이트 출퇴근 최적화

#### 👤 페르소나
**이름**: 이지우 (23세, 대학생)
**상황**: 카페 아르바이트 (시급 10,000원), 주 20시간 근무
**시급**: 10,000원

#### 😰 페인 포인트

**신규 아르바이트 시작 전날**
- 내일부터 새 카페에서 아르바이트
- 집에서 카페까지: 지하철? 버스? 택시?
- 수입: 월 80만원 (20시간 × 4주)

**내적 갈등**:
> "지하철이 제일 싸긴 한데... 환승 2번에 50분이나 걸려.
> 버스는 환승 없지만 70분...
> 택시는 25분인데 15,000원이나 해. 너무 비싸.
> 근데 매일 왕복 100분씩 쓰는 게 맞나?
> 월 40시간을 출퇴근에만 쓰네..."

**기존 의사결정**:
- 가격만 비교: 지하철(3,000원) 선택
- 현실: 매일 왕복 100분 소비
- 결과: 피로 누적 → 아르바이트 집중도 저하

#### 🔍 사용자 여정

**Step 1: 고민 시작 (전날 밤)**
- 상황: 지도 앱으로 경로 검색 중
- 감정: "매일 100분이 너무 아까운데..."
- 행동: "출퇴근 비용 비교" 검색

**Step 2: 앱 발견 & 다안 비교 사용**
```
[시급 입력]
- 10,000원 (본인 아르바이트 시급)

[다안 비교 모드 선택]
- "오, 3개 이상도 비교되네!"

[5가지 옵션 입력]
A: 지하철 (50분, 3,000원)
B: 버스 (70분, 2,500원)
C: 택시 (25분, 15,000원)
D: 지하철+택시 (35분, 10,000원)
E: 자전거+지하철 (40분, 2,000원)
```

**Step 3: 충격적인 결과**
```
결과:
1위: 🥇 E(자전거+지하철) - 총 8,667원
2위: 🥈 D(지하철+택시) - 총 15,833원
3위: 🥉 C(택시) - 총 19,167원
4위: A(지하철) - 총 11,333원
5위: B(버스) - 총 14,167원

충격:
- "헐, 지하철이 4등?"
- "자전거 조합이 제일 싸네?"
- "버스가 제일 비싸?"
```

**Step 4: 상세 분석 (계산식 확인)**
```
E안 상세 (자전거 15분 + 지하철 25분 = 40분)
- 직접 비용: 2,000원 (지하철만)
- 시간 비용: 10,000원/60분 × 40분 = 6,667원
- 총비용: 8,667원

vs

A안 상세 (지하철 환승 2번 = 50분)
- 직접 비용: 3,000원
- 시간 비용: 10,000원/60분 × 50분 = 8,333원
- 총비용: 11,333원

차이: 2,666원/일
```

**Step 5: 의사결정 & 실행**
- 결정: E안 (자전거+지하철) 선택
- 추가 행동: 중고 자전거 구매 (10만원)
- 계산: 투자 회수 = 10만원 ÷ (2,666원 × 2회) ≈ 19일

**Step 6: 1주일 후 평가**
```

- 매일 왕복: 80분 → 건강도 챙김
- 일일 비용: 11,333원 → 8,667원 (2,666원 절약)

- 체감: "10분만 더 투자하면 2,700원 벌고, 운동까지!"

추가 효과:
- 건강: 매일 자전거 30분 운동
- 정시성: 지하철 혼잡 피해 정시 도착
- 만족도: ★★★★★
```

| 페인 포인트 | 해결 |
```
