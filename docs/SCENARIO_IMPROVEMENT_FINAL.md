## 개선 전략

### 현재 문제
- 계산 결과만 표시 (여정 없음)
- 페인 포인트 불명확
- 맥락 부족

### 개선 방향
- 사용자 여정 3-4단계로 간결하게
- 페인 포인트 명시 (Before/After 표)
- 구체적 배경 추가
- 감정적 표현 제거

## 개선된 시나리오 1: 마트 vs 편의점

```markdown
### 시나리오 1: 퇴근 후 장보기

#### 배경
- **사용자**: 김민준 (32세, IT 직장인, 시급 20,000원)
- **시간**: 오후 9시 야근 후 퇴근
- **상황**: 냉장고 우유 부족, 내일 아침 커피용
- **장소**: 집 앞 (편의점 5분 vs 마트 25분)

#### 페인 포인트
1. **불확실성**: 가격 차이(700원)만으로 판단하기 어려움
2. **시간 가치 무인식**: "25분 정도는 괜찮겠지" (시간 비용 미계산)
3. **의사결정 피로**: 매일 반복되는 동일한 고민
4. **사후 후회**: 마트 다녀온 후 "시간이 너무 아까웠다" 불만족

#### 사용자 여정

| 단계 | 행동 | 소요 시간 | 결과 |
|------|------|-----------|------|
| 1. 문제 인식 | 편의점 앞에서 고민, 앱 검색 | 30초 | 앱 발견 |
| 2. 정보 입력 | 시급 20,000원, A(편의점 5분, 3,500원), B(마트 25분, 2,800원) | 1분 | 입력 완료 |
| 3. 결과 확인 | A: 5,167원, B: 11,133원, 차액 5,966원 | 10초 | 편의점이 유리 |
| 4. 의사결정 | 편의점 선택 | 즉시 | 확신 확보 |

**총 소요**: 1분 40초

#### 페인 포인트 해결

| 문제 | 해결 | 측정 |
|------|------|------|
| 불확실성 | 객관적 수치 제공 | 의사결정 확신도 40% → 95% |
| 시간 가치 무인식 | 시간 비용 가시화 (8,333원) | 시간 인식률 0% → 100% |
| 의사결정 피로 | 1분 40초 만에 결론 | 고민 시간 5분 → 1.5분 |
| 사후 후회 | 근거 명확 | 만족도 60% → 95% |

#### 결과
- **즉시 효과**: 5,966원 절약, 20분 절약
- **장기 효과**: 동일 상황 발생 시 즉시 결정, 의사결정 패턴 개선
```

## 개선된 시나리오 2: 직접 요리 vs 배달

```markdown
### 시나리오 2: 저녁 식사 준비

#### 배경
- **사용자**: 박수연 (38세, 전업 주부, 시급 환산 15,000원)
- **시간**: 오후 5시, 아이 둘 학원 픽업 후
- **상황**: 아이들 배고픔, 재료 있으나 조리 1시간 필요
- **제약**: 배우자는 배달비 부정적 ("너무 비싸")

#### 페인 포인트
1. **죄책감**: 배달 주문 시 "돈 낭비" 느낌
2. **자기 가치 절하**: 가사 노동을 무가치하게 평가
3. **가족 갈등**: 배우자와 소비 기준 차이
4. **에너지 소진**: 피로한 상태에서 1시간 요리

#### 사용자 여정

| 단계 | 행동 | 소요 시간 | 결과 |
|------|------|-----------|------|
| 1. 갈등 상황 | 부엌에서 배달비 vs 요리 시간 고민 | 5분 | 스트레스 |
| 2. 앱 사용 | 단톡방 친구 추천으로 앱 접속, 시급 15,000원 입력 | 1분 | 입력 완료 |
| 3. 계산 확인 | A(요리): 23,000원, B(배달): 19,250원 | 10초 | 배달이 3,750원 유리 |
| 4. 배우자 설득 | 계산 결과를 보여주며 배달 정당성 입증 | 1분 | 동의 확보 |

**총 소요**: 7분 10초

#### 페인 포인트 해결

| 문제 | 해결 | 측정 |
|------|------|------|
| 죄책감 | 배달이 3,750원 더 경제적임을 증명 | 죄책감 제거 |
| 자기 가치 절하 | 시간 비용 15,000원 가시화 | 자기 인식 개선 |
| 가족 갈등 | 객관적 데이터로 합의 도출 | 갈등 0건 |
| 에너지 소진 | 요리 대신 아이들과 시간 | 만족도 상승 |

#### 결과
- **즉시 효과**: 비용 3,750원 절약, 시간 55분 절약
- **장기 효과**: 가사 노동 가치 인정, 의사결정 기준 공유, 재사용 주 3회
```

## 개선된 시나리오 3: 출퇴근 방법

```markdown
### 시나리오 3: 아르바이트 출퇴근 최적화

#### 배경
- **사용자**: 이지우 (23세, 대학생, 시급 10,000원)
- **상황**: 신규 카페 아르바이트 시작 (주 20시간)
- **거리**: 집에서 카페까지 8km
- **제약**: 교통비 절약 필요, 하루 왕복 2회

#### 페인 포인트
1. **선택지 과다**: 지하철, 버스, 택시, 조합 등 5가지 이상
2. **장기 비용 불투명**: 한 달 교통비 총액 계산 어려움
3. **시간 vs 비용**: 싼 방법이 시간이 오래 걸림
4. **매일 반복**: 잘못 선택하면 한 달 내내 비효율

#### 사용자 여정

| 단계 | 행동 | 소요 시간 | 결과 |
|------|------|-----------|------|
| 1. 경로 검색 | 지도 앱으로 5가지 경로 확인 | 10분 | 혼란 가중 |
| 2. 다안 비교 | 앱에서 3~5개 선택지 비교 모드 사용 | 2분 | 5가지 입력 |
| 3. 결과 분석 | E(자전거+지하철) 8,667원으로 1위 | 30초 | 최적 경로 발견 |
| 4. 투자 결정 | 자전거 10만원 구매, 19일 후 회수 계산 | 1분 | 구매 결정 |

**총 소요**: 13분 30초

#### 페인 포인트 해결

| 문제 | 해결 | 측정 |
|------|------|------|
| 선택지 과다 | 5가지를 1분 만에 비교 | 의사결정 시간 10분 → 2분 |
| 장기 비용 불투명 | 일일 절약 2,666원 × 20일 = 53,320원/월 | 월간 비용 가시화 |
| 시간 vs 비용 | 총비용 기준으로 명확히 비교 | 최적 선택 도출 |
| 반복 비효율 | 한 번 결정으로 한 달 효율 확보 | 효율 향상 |

#### 결과
- **즉시 효과**: 일일 2,666원 절약, 자전거 투자 회수 기간 19일
- **장기 효과**: 월 53,320원 절약, 건강 개선 (자전거 30분/일)
```
