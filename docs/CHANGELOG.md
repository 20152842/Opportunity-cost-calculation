## v0.1 — 초기 설계

### 핵심 목표
- 사용자의 시급을 기준으로 선택지 A/B의 **총비용(Total Cost Index)**을 계산·비교하고,
- 결과와 함께 **근거(시간비용/직접비용 분해 + 수식)**를 제공하여 의사결정의 확신을 높인다.

### 핵심 기능(MVP)
1) 시급 입력 → 분당 가치 자동 계산  
2) 선택지 A/B 입력: 소요시간(분), 직접비용(원)  
3) 결과 출력: A 총비용, B 총비용, 추천 선택, 차액(원)  
4) 근거 제공: 직접비용/시간비용 분해 + 계산식 토글  
5) Total Cost Index(=총비용)로 비교 결과를 직관적으로 표시  
6) 프리셋 예시 제공(온보딩)

### 계산 모델(정의)
- 총비용(TC):  
  $$TC = C + \left( \frac{W}{60} \times T \right)$$  
  - W: 시급(원/시간), T: 소요시간(분), C: 직접비용(원)
- 소수점 처리: **원 단위 절삭(floor)**

### 엣지 케이스(초기 포함 범위)
- 시급 0/음수/문자 입력 차단(오류 메시지)
- 시간 0분 허용(시간비용 0)
- 비용 0원 허용(무료 옵션)
- 총비용 동일 시 “차이 없음(동일)” 표시
- 비현실적 큰 값 입력 시 경고(계산은 유지)
- 단위 혼동(시/분) 예방(UI 예시 또는 변환 지원)
